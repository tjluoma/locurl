
This script takes one argument, an URL, and looks for a redirect via the Location: header.

It will also parse Google result links such as this:

http://www.google.com/url?sa=t&rct=j&q=macdrifter&source=web&cd=1&ved=0CB0QFjAA&url=http%3A%2F%2Fwww.macdrifter.com%2F&ei=GlHZTrPBMJOosALWxqnvDQ&usg=AFQjCNGao2CZKtVtgsSQO046Pgm0oC3ZkA

and give you their destination:

http://www.macdrifter.com/

For more details, see

http://luo.ma/locurl
